# BTL-Python
Sản phẩm demo game Python bằng thư viện pygame

## Cách chơi
- Clone repo về, mở bằng pycharm hoặc các IDE khác
- Run and play
- Người chơi dùng 4 phím mũi tên để di chuyển nhân vật đến vị trí đích.
